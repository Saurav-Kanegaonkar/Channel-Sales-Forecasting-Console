# Analysis Plan

1. Generate channel partner, daily sales, forecast, promotion, compensation, territory, and source-event data.
2. Aggregate dealer demand by month and compare submitted forecast units with actual units.
3. Calculate recent demand momentum and inventory coverage for each partner.
4. Summarize promotion ROI and open compensation exposure by partner and region.
5. Add territory capacity pressure and source-system quality penalties.
6. Score each partner and create a ranked priority queue.
7. Convert the ranked queue into executive-ready actions and dashboard surfaces.

## Review questions

- Which channel partners need action before the monthly readout?
- Is the action a forecast reset, coverage change, promotion decision, compensation review, or data reconciliation?
- Which regional territories have enough capacity to execute the recommended action?
- Which recommendations should be withheld until source data is reconciled?
